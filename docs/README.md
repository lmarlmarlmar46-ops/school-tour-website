# Virtual School Tour Website

A comprehensive promotional website for school tours featuring interactive virtual walkthroughs, campus maps, user authentication, and more.

## 🎯 Features

### Core Features
- **Home Page** - Welcome page with hero section, facility showcase, and newsletter signup
- **Virtual Tour** - Interactive 360° walkthrough of all 5 floors with images and facility details
- **Campus Map** - Interactive floor plan with room locations and address information
- **Login/Register System** - Complete user authentication with local storage
- **User Dashboard** - Personal profile with tour progress tracking
- **About Us** - School information, mission, and vision
- **Contact Us** - Contact form and information
- **FAQ** - Frequently asked questions with expandable answers

### Advanced Features
- **Search with Auto-Suggest** - Smart search for rooms and facilities
- **User Profile Dashboard** - Track tour progress, time spent, and saved locations
- **Pricing & Comparison Tables** - Tour package comparison (Basic, Premium, In-Person)
- **Notification Center** - Real-time notifications with bell icon
- **System Status Page** - Live system status monitoring
- **Knowledge Base / Help Center** - Comprehensive help documentation
- **Custom 404 Error Page** - User-friendly error handling
- **Privacy & Terms Pages** - Legal documentation
- **Newsletter Signup** - Email subscription feature

### Interactive Elements
- Responsive navigation with mobile menu
- Smooth scrolling and animations
- Interactive floor selection in virtual tour
- Expandable FAQ items
- Password strength indicator
- Form validation
- Social login options

## 🚀 Getting Started

1. **Open the website**
   - Simply open `index.html` in any modern web browser
   - No server setup required - works with local files

2. **Create an account**
   - Click "Register Now" or navigate to `register.html`
   - Fill in your information (data stored in browser's localStorage)
   - Login with your credentials

3. **Explore the campus**
   - Start the virtual tour from the home page
   - Navigate through different floors
   - Use the search feature to find specific facilities
   - Check out the interactive campus map

## 🛠️ Customization

To customize the website:
1. Edit color variables in `styles.css` (`:root` section)
2. Update school information in HTML files
3. Replace placeholder images with actual photos
4. Modify facility details in `tour.html`
5. Update contact information across pages

## 📝 Notes

- Images are from Unsplash (replace with actual school photos)
